  7  * /_connect

 22  * /dbname/collection/_insert - insert request
 23  *   docs=<an array of objs to insert>
 24  *   opts=options
 25  * /dbname/collection/_delete - delete request
 26  *   criteria=<criteria for deletion>
 27  *   opts=options
 28  * /dbname/collection/_update - update request
 29  *   criteria=<criteria for update>
 30  *   newobj=<update content>
 31  *   opts=<options>
 32  *   
 33  * Special ops:
 34  * /_dbs - list of databases
 35  * /dbname/_cmd - database command on db dbname
 36  * /_cmd - database command on admin db
