move

Object move(<object> criteria, <string> to, callback)
    Move a chunk of this collection to a new shard. Use the criteria parameter to find the chunk to move.
    Parameters:
      criteria - must contain "find" field which matches at least one document in the chunk to move.
      to - the shard to which to move the chunk, e.g., "localhost:27019"
      function - } [callback] optional function to call when a response is received.
    Returns:
      undefined
    Throws:
      - Exception if criteria doesn't contain "find" or "middle" fields or callback is not a function

shard

Object shard(<object> key, unique, callback)
    Enables sharding on this collection. The collection's database must have sharding enabled before it can be enabled on the collection.
    Parameters:
      key - the field or fields on which to shard, e.g., {"x" : 1}
      boolean - } [unique] if the shard key should be unique
      function - } [callback] optional function to call when a response is received.
    Returns:
      undefined
    Throws:
      - Exception if callback is not a function

split

Object split(<object> criteria, callback)
    Split this collection into two chunks on a single shard. Use MongooseCollection.move to move one of the chunks to a new shard.
    Parameters:
      criteria - must contain "find" OR "middle" field which gives the split criteria.
      function - } [callback] optional function to call when a response is received.
    Returns:
      undefined
    Throws:
      - Exception if criteria doesn't contain "find" or "middle" fields or callback is not a function
